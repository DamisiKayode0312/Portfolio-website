LocIT

A location-tracking necklace with facial recognition and geofencing capabilities designed to enhance personal safety and security.

Technologies: Python, Fusion 360, Figma, Raspberry PI, Eagle

Details: This project involved designing a wearable device that tracks the user's location and sends alerts if they enter or exit a predefined area. The mobile app was designed using Figma and developed with FlutterFlow.

Raspberry Pi Weather Station

A weather monitoring station that logs environmental data with 95% accuracy, integrated with AWS for cloud-based data storage.

Technologies: Python, AWS, Raspberry Pi

Details: This project involved building a weather station that collects temperature, humidity, and pressure data. The data is stored in AWS for real-time monitoring and analysis.

Red Light, Green Light Game

An interactive version of the Squid Game-inspired Red Light, Green Light game using Arduino and motion sensors.

Technologies: Arduino, C++

Details: This project involved creating a game that uses motion sensors to detect player movement. The game mechanics were programmed in C++, and the hardware components included LEDs, buzzers, and an LCD screen.
